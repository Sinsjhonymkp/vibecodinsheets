# VibeCodeSheets — лендинг Angular-разработчика

Одностраничный лендинг на HTML, CSS и JavaScript. Сборка через [Vite](https://vitejs.dev/).

## Запуск

```bash
npm install
npm run dev
```

Откройте в браузере: [http://localhost:5173](http://localhost:5173).

## Сборка

```bash
npm run build
```

Результат в папке `dist/`.

## Превью продакшн-сборки

```bash
npm run preview
```

## Структура

-   `index.html` — разметка лендинга
-   `src/main.js` — точка входа и логика (меню, скролл, год в футере)
-   `src/style.css` — стили

Секции: Hero, Обо мне, Навыки, Контакты. В блоке контактов можно подставить свои ссылки (GitHub, Telegram, Email).
